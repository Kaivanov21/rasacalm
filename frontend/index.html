Rasa Secretary Chatbot
Rasa Secretary Chatbot simple Chatbot using Rasa framework
Features
Answer FAQs Answer FAQ questions about Computer Science Department, implemented using rules and “Chitchat and FAQs” feature
Provide teacher contact details Provide teacher contact details based of their surname, implemented using form and custom actions
Answer out-of-scope questions with ChatPGT Handle out-of-scope questions using call to ChatGPT API using custom actions.
